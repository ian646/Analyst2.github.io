Excel
Attendance Dashboard

Project which required an Excel dashboard
accessible to all managers, tutors and support staff
with visuals which clearly identified patterns and trends in the data.

Setting up the Dashboard! The first step was designing a dashboard which would clearly show the important KPIs in an engaging and colourful manner. Arranging visualisations, e.g. bar charts, line graphs and dials took the majority of my time. Slicers and filters were added so the user could view data for different months and even years.

Calculations needed to determine student percentage attendance. In this way I was able to produce mutiple charts showing ethnicity, gender, disability metrics and much more.

Cleaning and importing the data! Firstly, it is important to ensure the imported data is clean and I connected to the data sources (which in this case was an Excel csv. file). Of course before connecting to the data siurce it is best practice to transform and clean the data at source.
